## Ícones no sistema

Ícones são representações gráficas de um significado. Eles podem ser usados ​​para expressar ações, estados e até mesmo para categorizar dados. Os ícones são usados para aumentar a compreesão dos usuários e diminuir a carga cognitiva para uma ação, comando ou seções.

De modo geral, os ícones devem ser usados ​​em combinação com rótulos significativos para auxiliar os usuários a reconhecer rápidamente o conteúdo de forma acessível. No entanto, pode-se utilizar ícones sem acompanhamento de texto visível caso o significado já seja um padrão reconhecido pelos usuários, mas mesmo assim deve-se utilizar a legenda em um tooltip para garantir o entendimento e uma boa usabilidade.

## Construção

Para a construção dos nossos ícones foi definido o auxílio de uma caixa que mantém a coerência de tamanho para todos os ícones, além de auxiliar na aplicação de layouts e modificações dos tamanhos. A grid padrão foi montada em um espaço de 24 pixels, onde os elementos se encaixam na estrutura com a margem obrigatória mínima de 2 pixels horizontal ou vertical.
