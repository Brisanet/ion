# Tag

No Ion, o padrão de uma `tag` é sua estilização neutra com borda, mas o mesmo pode receber
os seguintes parâmetros:

- `outline` Um boolean que define se irá ter borda(por padrão vem verdadeiro, trazendo a `tag` com borda)
- `label` A mensagem que será mostrada na `tag`
- `status` Que define tipos prioritários para a `tag`
- `color` Que pode passar uma cor customizada para a `tag`
- `icon` Que passa um icone customizado

## Texto(label)

A partir do atributo label o usuário irá informar qual mensagem será mostrada na tag

## Cores customizadas(colors)

Por meio do parâmetro `color` pode ser passado cores customizadas para a tag

- Esse atributo aceita cores em hexadecimal.
- Caso seja passado juntamente com o `status` a tag irá priorizar o status e desconsiderará a cor passada

## Fundo customizado (background)

Por meio do parâmetro `background` pode ser passada uma cor de fundo customizada para a tag.

- Esse atributo aceita cores em hexadecimal.
- Caso seja passado juntamente com o `status`, a tag irá priorizar o status e desconsiderará a cor de fundo passada.
- Se a propriedade `color` for usada em conjunto, ela definirá a cor do texto e da borda, enquanto `background` definirá a cor de fundo.
