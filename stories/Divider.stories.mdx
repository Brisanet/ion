# Divider

<Canvas>
  <Story name="type: default divider">
    {{ template: `<ion-divider> </ion-divider>` }}
  </Story>
</Canvas>

No Ion, a orientação padrão do divisor é `horizontal`. Mas fornecemos 2 tipos de orientação:

- `horizontal`. Indica que o divisor está na orientação horizontal.

<Canvas>
  <Story name="type: div with horizontal divider">
    {{
      template: `<div style="width: 150px"><ion-divider orientation="horizontal"> </ion-divider></div>`,
    }}
  </Story>
</Canvas>

O tamanho do divisor `horizontal` será definido pelo elemento HTML em que o mesmo está contido.

<Canvas>
  <Story name="type: horizontal divider">
    {{ template: `<ion-divider orientation="horizontal"> </ion-divider>` }}
  </Story>
</Canvas>

- `vertical`. Indica que o divisor está na orientação vertical.

<Canvas>
  <Story name="type: vertical divider">
    {{
      template: `<ion-divider orientation="vertical"> </ion-divider>`,
    }}
  </Story>
</Canvas>

O tamanho do divisor `vertical` será definido pelo elemento HTML em que o mesmo está contido.

<Canvas>
  <Story name="type: div with vertical divider">
    {{
      template: `<div style="height: 150px"><ion-divider orientation="vertical"> </ion-divider></div>`,
    }}
  </Story>
</Canvas>

No Ion, o divisor possui 3 tipos.

- `solid`. Divisor sólido.

<Canvas>
  <Story name="type: solid divider">
    {{ template: `<ion-divider type="solid"></ion-divider>` }}
  </Story>
</Canvas>

- `dashed`. Divisor tracejado.

<Canvas>
  <Story name="type: dashed divider">
    {{ template: `<ion-divider type="dashed"></ion-divider>` }}
  </Story>
</Canvas>

- `text`. Divisor com uma `label` em seu centro.

<Canvas>
  <Story name="type: text divider">
    {{ template: `<ion-divider type="text" label="Label"></ion-divider>` }}
  </Story>
</Canvas>

Nota-se: Divisores do tipo `text` só estarão disponíveis na `orientation: horizontal`,
porém não será necessário passar tipo `orientation`, pois `horizontal` será o tipo
padrão desse divisor.
