# Divider

No Ion, a orientação padrão do divisor é `horizontal`. Mas fornecemos 2 tipos de orientação:

- `horizontal`. Indica que o divisor está na orientação horizontal.
- `vertical`. Indica que o divisor está na orientação vertical.

<Canvas>
  <Story name="type: horizontal divider" args={{}}>
    {Divider.bind({})}
  </Story>
</Canvas>

<Canvas>
  <Story name="type: vertical divider">
    {{ Divider: `<ion-divider orientation="vertical"></ion-divider>` }}
  </Story>
</Canvas>

O Ion, Suporta 2 tipos de linha para o divisor. A `continuous` e a `dashed`:

- `continuous`. Divisor sólido.
- `dashed`. Divisor tracejado.

<Canvas>
  <Story name="type: continuous divider">
    {{ Divider: `<ion-divider type="continuous"></ion-divider>` }}
  </Story>
</Canvas>

<Canvas>
  <Story name="type: dashed divider">
    {{ Divider: `<ion-divider type="dashed"></ion-divider>` }}
  </Story>
</Canvas>

O Ion, também oferece suporte para o tipo `text`. Onde é possível inserir uma `string` ao divisor.
